# Era Roadmap

This roadmap reflects where I want Era to go over time.

The project is still in its early stages, and many ideas are experimental. Rather than promising exact release dates, this roadmap represents the general direction I want the project to move in.

Priorities may change as I continue learning, experimenting, and receiving feedback from the community.

---

# Current Focus

These are the areas I'm actively working on.

- Improve overall AI response quality
- Better prompt engineering
- Better desktop user experience
- Refactor and clean the codebase
- Improve project architecture
- Expand documentation

---

# Short-Term Goals

These features are planned once the current foundation becomes stable.

## Chat

- Edit previous messages
- Regenerate responses
- Better conversation management
- Improved Markdown rendering
- Export conversations in multiple formats

## Memory

- Better memory extraction
- Memory editing
- Semantic memory search
- Memory importance scoring
- Automatic memory cleanup

## Documents

- Better PDF support
- DOCX support
- Improved document parsing
- Better chunking
- Faster document indexing

## User Experience

- Better animations
- Keyboard shortcuts
- Drag and drop document support
- Better accessibility
- Improved desktop layout

# Long-Term Vision

These ideas represent what I ultimately want Era to become.

## Local AI Workspace

Instead of being just another chat application, I want Era to become a complete offline AI workspace where users can:

- Chat naturally
- Analyze documents
- Generate code
- Remember long-term information
- Organize knowledge
- Automate repetitive tasks

while keeping everything local.

## Agents

Future agent capabilities may include:

- File management
- Project assistance
- Local automation
- Multi-step reasoning
- Background tasks

## Voice

- Speech-to-text
- Text-to-speech
- Voice conversations
- Offline voice models

## Vision

- Image understanding
- OCR
- Screenshot analysis
- Diagram explanation

---

# Beyond v1.0

Some ideas are intentionally far in the future.

- Plugin system
- Local API
- Extensions
- Multi-workspace support
- Local knowledge graph
- Team collaboration (optional)
- Cross-device synchronization (privacy-first)

---

# Guiding Principles

No matter how Era evolves, these ideas should remain unchanged.

- Privacy comes first.
- Local-first whenever possible.
- Open source.
- Simple over complicated.
- Build a strong foundation before adding features.
- Learn by building and improve continuously.

---

This roadmap isn't a fixed promise.

It's a snapshot of where I believe Era can grow over time.
